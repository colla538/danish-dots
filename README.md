# danish-dots

My Dotfiles 

## DEPENDENCIES
- Pacman
  - hyprland & wayland (WM)
  - networkmanager (for wifi interface)
  - hyprlock (lockscreen)
  - ttf-meslo-nerd (fonts)
  - swaybg (wallpaper)
  - swaync (notification)
  - waybar (bar)
  - grim, slurp, swappy (screenshot)
  - nwg-bar (shutdown menu)
  - stow (dotfiles stow)
  - wofi (menu)
  - foot (main terminal)
- AUR
    - hyprfloat 
    - tty-clock (clock)

## Installation steps
1. clone repo
```sh
git clone https://github.com/colla538/dofiles.git
```

2. cd to `~/.dotfiles`

```sh
cd ~./dotfiles
```

4. Use `stow .` to put the config in their places

```sh
stow .
```
